📌 NLP Text Preprocessing Pipeline

This project implements an advanced text preprocessing pipeline for Natural Language Processing (NLP) tasks. It focuses on cleaning noisy text data and preparing it for further analysis like sentiment analysis, classification, or machine learning models.

🚀 Features
Convert text to lowercase
Remove numbers and unwanted characters
Remove URLs and email patterns
Handle repeated characters (e.g., soooo → soo)
Remove extra spaces
Filter very short words (while keeping important ones like "no" and "not")
Tokenization and sentence cleaning
Frequency analysis using Counter
Token-level analytics
Robust error handling for edge cases
🛠️ Technologies Used
Python 🐍
Regular Expressions (re)
Collections (Counter)


⚙️ How It Works
Input raw text
Clean and normalize text
Tokenize into words
Remove noise and unwanted elements
Output cleaned tokens and sentences

📊 Additional Analysis
Token count, unique words, and average length
Most and least frequent words
Noise detection in sentences
⚠️ Error Handling

The system safely handles:

Empty strings
Only emojis
Only numbers
🎯 Use Cases
Sentiment Analysis
Chatbots
Text Classification
Data Cleaning for ML models
